# Tyrell Cromoshuk Personal Site

Professional portfolio site for Tyrell Cromoshuk, positioned around Growth & E-Commerce Executive roles and senior consulting/advisory work.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Content

Most site content lives in `src/lib/content.ts`:

- proof metrics
- industries
- platform ecosystem
- services
- career timeline
- case studies

The contact form opens a pre-filled email by default. To post to a hosted form endpoint, set `NEXT_PUBLIC_CONTACT_FORM_ENDPOINT` in Vercel.

Use `COPY_GUARDRAILS.md` before adding or editing site copy.
